Install
------------------------------------------------------------------------------

``logingov_doc_example`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install logingov_doc_example

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade logingov_doc_example
